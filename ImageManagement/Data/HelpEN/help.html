Menu

Menu > File

Menu > File > New

Creates the basic *.im.json file and a folder for your images, you can import them with Menu > Images > Import

Menu > File > Open

Opens a *.im.json file and all associated images

Menu > File > Save

Saves the currently opened *.im.json file and all edited images

Menu > File > Save As

Saves the currently opened *.im.json file and all edited images to the given directory (full export)

JSON-Structure

{
  "Images": [], // List of all Images
  "Tags": {}, // List of all Tags
  "Characters": {} // List of all Characters
}

Image

{
  "path": "./<filename>",
  "alias": "",
  "tags": [], // list of the used tags keys
  "characters": [], // list of the used characters keys
}

Tag

"<key>": {
  "synonyms": [], // list of all Synonyms for this Tag (strings)
  "description": "", // description for this tag 
}

Character

"<key>": {
  "name": "",
  "aliases": [], // list of all aliases (strings),
  "gender": "",
  "description": "",
  "date-of-birth": {
    "day": 1,
    "month": 1,
    "year": 2000
  }
}
